# SDKWork Plugins

This directory stores checked-in repository or application plugins for $name.

Plugins must follow $specPath/SDKWORK_WORKSPACE_SPEC.md. Installable plugins must provide .codex-plugin/plugin.json and must document contributed skills, tools, apps, scripts, or MCP servers.

Plugins must not vendor unrelated toolchains, generated SDK output, runtime databases, logs, cache, secrets, API keys, private certificates, or user-private files.
